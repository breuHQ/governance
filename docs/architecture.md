## Overview Of The Key Components

### Project Build Squad 
These are the Developers, Teams, Admins and Maintainers which can overlap

### User File

Sample File Example - 

```JSON

email: john_doe@company.com
recovery_email: john.doe@email.com
name:
  first_name: John
  last_name: Doe

gsuite_user: true

# Google Workspace
gsuite:
  groups:
    - name: team
      role: member
# Github
github:
  username: john
  teams:
    - name: Project A
 role: member

```
